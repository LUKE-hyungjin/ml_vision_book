# Classical Computer Vision

Representative computer-vision methods from before deep learning became dominant.

## Characteristics

- Hand-crafted features (human-designed descriptors)
- Strong interpretability (you can inspect each step)
- Works with relatively small datasets
- Usually lighter than modern deep models on CPU-only environments

## Main Method

- [SIFT & HOG](/en/docs/architecture/classical/sift-hog) - classical local/gradient feature pipelines
